# Hook System Overview

MoAI-ADK Hook 시스템은 개발 워크플로우에 커스텀 로직을 삽입할 수 있게 합니다.

## 사용 가능한 Hooks

1. **SessionStartHook**: 세션 시작 시 실행
2. **PreToolUseHook**: 도구 사용 전 실행
3. **PostToolUseHook**: 도구 사용 후 실행
