    moai_adk.utils.progress_tracker

    @FEATURE:PROGRESS-TRACKER-001 Progress tracking and display utilities for MoAI-ADK installation @TASK:UI-PROGRESS-001 Provides clean, professional progress display with color coding and customizable progress callbacks for different UI contexts

    This module handles: - Real-time progress visualization with progress bars - Multi-stage progress tracking for complex operations - Color-coded status indicators - Customizable progress callbacks for different UI contexts
