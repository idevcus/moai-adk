    moai_adk.core.quality.guideline_checker

    Guideline compliance checker for MoAI-ADK.

    Validates Python code against TRUST 5 principles development guidelines: - Function length ≤ 50 LOC - File size ≤ 300 LOC - Parameters ≤ 5 per function - Complexity ≤ 10 per function

    @FEATURE:QUALITY-GUIDELINES Guideline compliance validation system @DESIGN:ARCHITECTURE-001 Improved structure with constants and utilities
